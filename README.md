# AI Engineering Master Live Coding #01

A set of increasing difficulty scripts were developed, in order to understand basics of LLMs harness (API calls with simple message, system prompts, temperature, max output tokens, conversation, error handling, streaming, basic chatbots).

# Requirements

* Python 3.12
* OpenAPI API key to insert in .env

## Quickstart

```powershell

# create python virtual env
python -m venv .venv

# activate virtual env
cd .venv
\scripts\activate

# install dependencies
pip install -r requirements.txt

# to run a file
python 01_first_call.py

```
